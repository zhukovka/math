Concepts and Vocabulary

  1. a = a

  2. if a = b then b = a

  3. if a = b and b = c then a = c

  4. if a = b then we may substitute b for a

  5. a + b = b + a

    a × b = b × a

  6. a + (b + c) = (a + b) + c = a + b + c

    a × (b × c) = (a × b) × c = a × b × c

  7. (a + b) × c = a × c + b × c

  8. 0 + a = a + 0 = a

    a × 1 = 1 × a = a
